# ADR-001: Initial project setup

**Status:** Active
**Date:** 2024-11-09
**Authors:** Muhammad Adeel
**Commit:** 
**Repository:** https://github.com/muhammadadeel147/Kanbas-app

---
## Context

The project requires an initial setup to start building the application. The goal is to quickly set up the project structure.

**Problem Statement:**
Setting up the initial project structure is crucial for starting the development process.

---
## Decision Options Considered

| Option | Description | Pros | Cons | Why Rejected/Accepted |
|--------|-------------|------|------|----------------------|
| Create a new React project | Use create-react-app to generate the initial project structure | Fast project setup, well-structured codebase | Limited customization options | Accepted |

---
## Decision Made

Create a new React project using create-react-app.

---
## Reasoning

This option was chosen because it provides a fast and well-structured way to set up the project. The create-react-app tool generates a basic project structure, including configuration files and a sample application.

**Key Factors:**
1. Speed of setup
2. Quality of the generated codebase
